# Tressure Hunt Web Portal

Web Portal Allow You to conduct a competition for your collage competition


# TechStack :

1. React JS
2. Redux
3. Tailwind CSS

# Live Project Link : 
https://tressure-hunt-frontend.vercel.app/

# Run Project :

1. Clone Repository
```
git clone https://github.com/Yadhnesh1234/tressure-hunt-frontend/
```
2. Install Dependency
```
npm i
```
3. Run Project
```
npm start
```
